In my last post I was made a mistake in regards to [UpNote](https://getupnote.com/). I stated it supports markdown and it does but not as seemelessly as [Obsidian](https://obsidian.md/).
Here is a quote from the developer on [reddit](https://www.reddit.com/r/UpNote_App/comments/14b794n/comment/jokfbl9/?utm_source=share&utm_medium=web3x&utm_name=web3xcss&utm_term=1&utm_content=share_button). 
> In UpNote the note is stored in HTML format, not Markdown and it’s probably not useful to show the underlying HTML code to user. You can still export the note to various formats if you need to. 

I am still going to try out both Obsidian and UpNote to see which one meets my needs best.  Or I may end up using both haha.
