Just getting setup with my humble homelab. It starts with the firewall which runs on a HP Compaq Pro 6300. I bought the pc locally for $100 and so far it runs pfSense without issue. 

I recently bought a managed switch, my first, from Ubiquiti. The UniFi Switch Lite 8 PoE which can be found [here][1]. I want to buy an access point or two but Ubiquiti are currently sold out of the wifi 6 version so I will wait on that for later. I want to put all of my IOT devices on their own wifi VLAN. I have several of these type of devices which includes a thermostat, smart plug, smart lock, smart doorbell, and so on.

I have the Unifi controller running on a Raspberry Pi along with Pi-hole. I also run another instance of  Pi-hole in a docker container on my Synology NAS as a backup.

The server this blog runs on is on a VLAN so it can be segregated from the rest of my LAN. I have it running in a docker container on Fedora Linux. I&#8217;m also running a Channels DVR server. I followed a youtube video to get VLAN&#8217;s setup right.
